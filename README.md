# LLM-Memory-Retention-Demo
연구 보안으로 인해 실제 재활 훈련 질문은 공개하지 않고, 생성형 AI로 대체 생성한 질문 세트를 기반으로 LLM 기억 유지 성능 평가 및 개선 구조를 공개용 데모로 재구성하였습니다.

# Purpose
본 프로젝트는 다음 내용을 중심으로 구성됩니다.
- 장기 대화에서의 기억 유지 성능 평가
- 요약 기반 메모리 구조
- 사람 피드백 기반 요약 보정 구조
- 턴 수에 따른 메모리 성능 비교

# 코드 별 역할

demo_memory_eval.py 는 LLM의 기억 소실 현상을 평가하기 위한 코드입니다. 
demo_summary_memory.py 는 LLM에 이전 대화 로그에 대한 요약이 포함되어 있을 때 기억 성능을 평가하기 위한 코드입니다. 
demo_human_feedback.py 는 LLM 대화 로그 요약에 사람이 직접 요약 성능을 평가하기 위한 코드입니다.
